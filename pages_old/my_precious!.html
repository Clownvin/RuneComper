My Precious! is an achievement which requires the player to return Gregg 'groggy' herring's ring to Gregg 'Groggy' Herring.

Update history[edit | edit source]

This information has been compiled as part of the update history project. Some updates may not be included—see here for how to help out!
  • patch 16 January 2023 (Update):
    • Became a requirement for Master quest cape.
  • update 5 March 2018 (Update):
    • Added to game.

Trivia[edit | edit source]

  • The name is a reference to Gollum's name for the One Ring from The Lord of the Rings.
