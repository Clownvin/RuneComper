Making Learning Fun! is an achievement that requires the player to enter the second level of the Stronghold of Security.

This information has been compiled as part of the update history project. Some updates may not be included—see here for how to help out!

    • Added to game.

Trivia[edit | edit source]

  • This achievement used to be a task prior to the release of the Achievements System on 18 April 2017.
