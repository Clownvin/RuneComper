Varrock Set Tasks - Medium
Release date 18 April 2017 (Update)
Members Yes
Description Given by Reldo in Varrock Palace's library for completing all Medium Tasks in Varrock.
RuneScore 10
Category Area Tasks
Rewards
  • Varrock armour 2
  • Magic lamp worth 5,000XP in a skill of your choice.
  • The armour gains the following extra benefits:
    • +3% chance for double smelting mithril, adamant.
    • +3% chance for double ore mining mithril, adamantite, luminite.
    • +3% chance for double progress smithing mithril, adamant.
    • Further improved chance of receiving a skull sceptre piece in the Stronghold of Security.
    • Skull sceptres have additional charges if assembled while wearing the armour.
  • Naff will sell up to 32 battlestaves a day.
Requirements
  • See requirements.
Advanced data

Varrock Set Tasks - Medium is an achievement that is rewarded upon finishing the Medium Varrock achievements.
